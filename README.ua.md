## Про проєкт

Це початківський Rust проєкт. Мета отримати практичний досвід з:

- Структурами та енумами
- Парсингом аргументів командного рядка (`std::env::args`)
- Pattern matching
- Роботою з `Vec` та `String`

Без зовнішніх крейтів лише стандартна бібліотека.

## Використання

```bash
cargo run
```

Програма запускається в інтерактивному режимі просто вводиш команди і натискаєш Enter:

```
> add Купити хліб
> done 1
> remove 2
> edit 1 Купити молоко
```

### Команди

| Команда | Опис |
|---|---|
| `add [Назва]` | Додати нове завдання |
| `done [ID]` | Позначити завдання як виконане |
| `remove [ID]` | Видалити завдання |
| `edit [ID] [Нова назва]` | Змінити назву завдання |
